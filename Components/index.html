JS组件

登录验证 :

基础验证,未使用正则匹配复杂验证逻辑
验证逻辑: 用户名,手机号,密码长度限制, 邮箱必须包含 @ 等

标签tabs :

通过修改 tabs.js 用户配置项: tabList
可动态控制: 标签数量, 标签内容, 挂载的目标元素

下拉搜索 :

可搜索列表: [ 王二小, 李二狗, 赵四儿, 张三儿 ]
可模糊匹配，举例：三 → 张三儿
通过修改 dropdown.js 用户配置项:list, 可替换搜索列表

github搜索 :

Github Search

原生js请求api.github，可以搜索用户名，举例：1938s
注意: api.github 限制同一ip每分钟的请求数量，避免频繁发送请求。

弹出层 :

通过修改 popup.js 用户配置项: config
可动态控制: 弹窗弹出位置, 挂载的目标元素
